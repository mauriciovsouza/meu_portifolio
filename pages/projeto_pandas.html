Pandas para fazer extração de uma planilha base.

Problema de negócio:

    A empresa recebeu uma base de dados em excel.
    Uma planilha base, onde tem várias colunas com informações diversas, entre elas, nome repetida de Cidades. Há a necessidade de juntar as informações de cidade por cidade e extrair para arquivos separados.

Planejamento:

    Filtrar todas as cidades e fazer a extração para arquivos separados no excel.

Construção:

    Sabendo do poder do Pandas no python, utilizei engenharia de prompt com ChatGPT para criar um código que: abra um arquivo excel, filtre uma coluna, no meu caso a coluna MUNICÍPIO, e extraia cada node de cidade para um xlsx diferente.

Entrega:

    A ferramenta está finalizada, após alguns testes de extração e implementação da mesma no dia-a-dia, ela atende 100% do que precisamos.
