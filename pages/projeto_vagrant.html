O que é Vagrant?

    O Vagrant é uma ferramenta de código aberto que facilita a criação, configuração e gerenciamento de ambientes virtuais de desenvolvimento. Ele permite que desenvolvedores configurem máquinas virtuais de forma consistente e automatizada, garantindo que o ambiente de desenvolvimento seja idêntico ao ambiente de produção.
    Para criação de uma máquina virtual utilizando Vagrant, é necessário um arquivo contendo as configurações que a VM usará, esse arquivo é chamado de Vagrantfile.
    Nesse projeto utilizei um script que está dentro do arquivo provision.sh que contém alguns comando de instação de programas e pacotes, esses comandos são rodados de forma automatizada, sem haver a necessiade de quem está criando a VM logar na VM e fazer as instalações manuais.
